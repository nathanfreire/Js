FRAMEWORK

Um FRAMEWORK é um conjunto de ferramentas, bibliotecas e regras que facilita o desenvolvimento de software, fornecendo uma estrutura pronta para o programador. O FRAMEWORK define um "esqueleto" do projeto, onde você preenche com seu código, seguindo padrões já estabelecidos. Isso economiza tempo, evita a repetição de código e garante boas práticas.
Um FRAMEWORK é como montar um hamburguer em uma rede de fast food. Sem um FRAMEWORK, voce teria que moer a carne, assar o pao, preparar os ingredientes e criar sua propria receita de zero.

Array ou vetor é uma estrutura de dados indexada 
Array é uma alocação de variavel, com a diferença que ele pode armanezar vários dados.
O array trabalha com indices, o indice sempre inicia em 0.
Usos do array:
- Simplificar o código
- Estrutura de dados (manipular os dados - banco de dados).
